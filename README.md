# spring-ai-cookbook
本指南为 Java 开发者提供 Spring AI 开发的入门资源。Spring AI 是一个简化 AI 集成的强大框架，提供 Spring 友好的 API。指南涵盖聊天补全、文本到图像、音频转录等功能，并介绍如何结合主流 AI 模型（如 DeepSeek、Ollama）快速构建本地化服务，同时提供代码示例和最佳实践，助力开发者高效、安全地开发 AI 应用。
